6.S191 2018 Course Portal

This page will contain important information for the 2018 course including announcements, and key links.

Announcements:

- [Jan 20, 2018] To prepare for the first day of software labs please bring a laptop with Docker CE installed before lecture starts. To install docker please download the following file depending on your operating system: Ubuntu, Mac, Windows. Please complete this before the first lecture (Jan 29, 2018) -- if you have any issues please post on Piazza (see below).
To test your installation run docker run hello-world in your terminal command line.

Piazza:

https://piazza.com/class/jcp36n5pwq4wo
Please join and post to Piazza with any course related questions. We prefer this over email so other students can also see and benefit from the answer.

Help Queue:

http://deepqueue.herokuapp.com
During lab hours you can submit real-time tickets to be helped in person by logging into our help queue. Please use your Github account to login, if you don't have a Github please create one, or contact the course staff to create an account for you.
